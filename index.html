What Artemis is

A local assistant for the machine in front of you.

Artemis is being built as a private desktop companion that can listen, speak, summarize project state, expose local status, and preserve safe continuity across work sessions. Chat is one surface, not the whole product.

Privacy and safety

Local-first by default, action-safe by design.

Local models and local speech tooling are the default direction.

No tracking, analytics, remote scripts, remote images, or external fonts in this scaffold.

Private and no-memory modes remain product boundaries, not decorative labels.

Future computer control requires explicit approval, permissions, traces, and audits.

Roadmap

From local assistant to safe desktop command center.

  1. Now: voice, STT, TTS, Ollama routing, and terminal status.
  2. Now: dashboard bridge, memory policy, and project continuity.
  3. Sprint 19: original website positioning and static visual contract.
  4. Next: read-only app shell that renders safe local status.
  5. Later: approval-gated computer-action layer after safety review.
  6. Later: packaged local desktop build after the shell is stable.
